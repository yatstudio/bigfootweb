# Bigfoot Capital Website

Static bilingual website for Bigfoot Capital.

## Pages

- English: `index.html`
- 中文: `zh.html`

## Assets

- Bigfoot logo and mascot assets are stored in `assets/`.

## Local preview

Open directly in a browser:

```text
file:///Users/yat/bigfoot-capital-site/index.html
file:///Users/yat/bigfoot-capital-site/zh.html
```

Or run a local static server:

```bash
python3 -m http.server --bind 127.0.0.1 8765
```

Then visit:

```text
http://127.0.0.1:8765/
http://127.0.0.1:8765/zh.html
```
